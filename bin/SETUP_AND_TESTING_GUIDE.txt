# Peer Review Platform — Backend Setup, Testing & Frontend Guide
# ================================================================


## STEP 1 — MySQL Setup
-----------------------
1. Open MySQL Workbench or MySQL terminal
2. Run the file:  src/main/resources/schema.sql
3. This creates the database and all 7 tables automatically


## STEP 2 — Configure application.properties
--------------------------------------------
Open:  src/main/resources/application.properties

Change this line to your MySQL password:
    spring.datasource.password=your_mysql_password

If your MySQL username is not "root", change this too:
    spring.datasource.username=root


## STEP 3 — Run the Backend
----------------------------
Open terminal in the project folder and run:

    mvn clean install
    mvn spring-boot:run

Server starts at:  http://localhost:8080

## STEP 5 — Connect Frontend to Backend
========================================

In your React project, update Login.jsx and Register.jsx
to call the API instead of using localStorage.

--- Login.jsx (replace handleLogin) ---

  const handleLogin = async (e) => {
    e.preventDefault();
    try {
      const response = await fetch("http://localhost:8080/api/auth/login", {
        method: "POST",
        headers: { "Content-Type": "application/json" },
        body: JSON.stringify({
          email: formData.email,
          password: formData.password,
        }),
      });

      if (!response.ok) {
        const err = await response.json();
        alert(err.error || "Login failed");
        return;
      }

      const data = await response.json();
      localStorage.setItem("token", data.token);
      localStorage.setItem("currentUser", JSON.stringify({
        name: data.name,
        email: data.email,
        role: data.role,
      }));

      if (data.role === "admin") {
        navigate("/admin-dashboard");
      } else {
        navigate("/student-dashboard");
      }
    } catch (err) {
      alert("Server error. Is the backend running?");
    }
  };


--- Register.jsx (replace handleSubmit) ---

    } catch (err) {
      alert("Server error. Is the backend running?");
    }
  };


--- Helper: apiCall utility (create src/api.js in React) ---

  const BASE_URL = "http://localhost:8080";

  export function getToken() {
    return localStorage.getItem("token");
  }

  export async function apiCall(endpoint, method = "GET", body = null) {
    const headers = {
      "Content-Type": "application/json",
      Authorization: "Bearer " + getToken(),
    };

    const options = { method, headers };
    if (body) options.body = JSON.stringify(body);

    const res = await fetch(BASE_URL + endpoint, options);
    if (!res.ok) {
      const err = await res.json();
      throw new Error(err.error || "Request failed");
    }
    return res.json();
  }

## COMMON ERRORS & FIXES
==========================

ERROR: "Access Denied" (403)
  → You are using wrong role token.
  → Admin endpoints need admin token. Student endpoints need student token.

ERROR: "Could not create connection to database"
  → MySQL is not running. Start it first.
  → Wrong password in application.properties.

ERROR: CORS error in browser
  → Make sure app.cors.allowed-origins=http://localhost:5173 in properties.
  → React must run on port 5173 (default Vite port).

ERROR: "Table doesn't exist"
  → You forgot to run schema.sql in MySQL.

ERROR: JWT expired
  → Login again to get a new token.
